# NLISBuddy Website

![NLISBuddy Logo](public/favicon.ico)

**NLISBuddy** is a modern, premium-minimal landing page for the NLISBuddy mobile app — a tool to simplify livestock compliance and NLIS record management for Australian producers.

This repository contains the **React + Tailwind CSS source code** for the official NLISBuddy website, including a hero section with animated logo, feature highlights, download section with App Store & Google Play badges, a donation option, and a premium footer.

---

## **Features**

- **Hero Section:** Animated, floating logo with subtle glow and soft shadow  
- **Responsive Layout:** Fully mobile-friendly with Tailwind CSS  
- **Download Section:** App Store & Google Play badges linking to respective download pages  
- **Donate Option:** Functional PayPal / Stripe donation button (configurable)  
- **Footer:** Premium minimal footer with navigation, contact info, and subtle scroll animations  
- **Smooth Scroll:** Hero button links to the download section with smooth scrolling  
- **Modern Animations:** Implemented with Framer Motion for subtle entrance and hover effects  
- **Tailwind Design System:** Custom colors, shadows, and reusable components  

---

## **Tech Stack**

- **React** – Frontend framework  
- **Tailwind CSS** – Utility-first styling  
- **Framer Motion** – Animations and scroll effects  
- **JavaScript / JSX** – Component logic  
- **Optional:** Stripe / PayPal for donation button
